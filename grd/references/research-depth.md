# Research Depth Levels

Three tiers of research depth. Set during `/grd:new-project` questioning, recorded in PROJECT.md. Individual phases can override.

## Level 1: Orientation

**Trigger:** "What is X?" / unfamiliar topic / background context needed
**Source requirements:** 3-5 secondary sources (review articles, encyclopedias, textbook chapters)
**Search strategy:** Curated — find the best overview, not every source
**Critical analysis:** Summarize accurately. Identify major concepts and key figures.
**Quality indicators:** Accurate, well-organized, identifies the right sub-topics for deeper investigation
**Output:** Single note with broad coverage, pointers to deeper reading

## Level 2: Working Knowledge

**Trigger:** "How does X work?" / building on orientation / need to make decisions
**Source requirements:** 10-20 sources mixing primary and secondary. Must include ≥3 primary sources (original papers, not summaries of papers).
**Search strategy:** Targeted — identify seminal works, recent reviews, and 2-3 key debates
**Critical analysis:** Explain to a peer. Know the key authors. Understand why experts disagree where they do. Distinguish settled findings from active debates.
**Quality indicators:** Could teach a graduate seminar. Knows what they don't know. Cites primary sources for key claims.
**Output:** Multiple notes organized by sub-topic, with identified gaps

## Level 3: Expert Synthesis

**Trigger:** "What is the state of knowledge on X?" / doctoral quality / publishable output
**Source requirements:** 30-50+ primarily primary sources. Systematic search with documented inclusion/exclusion criteria. Citation network traced (who cites whom, intellectual lineage).
**Search strategy:** Systematic — documented search terms, databases searched, date ranges, inclusion/exclusion criteria. Multiple passes: breadth scan → targeted deep reads → citation chain following.
**Critical analysis:** Original synthesis with evaluative argument. Engage with methodology (study designs, sample sizes, effect sizes, replication status). Identify gaps in the literature. Position findings relative to competing frameworks. Distinguish between what evidence shows and what authors claim it shows.
**Quality indicators:** Novel connections between sources. Identifies genuine gaps. Engages with methodology, not just conclusions. A domain expert would learn something. Could withstand peer review.
**Output:** Comprehensive note set with executive summary, structured by theoretical framework or thematic analysis

## Depth Selection

- Depth is set during `/grd:new-project` questioning and recorded in PROJECT.md
- Individual phases can override (a Level 2 project may have one Level 3 phase for a critical question)
- Depth level propagates to researcher agents via the orchestrator prompt
- Source count minimums scale with depth level
- Critical analysis expectations scale with depth level

## Depth-Aware Behaviors

| Behavior | Level 1 | Level 2 | Level 3 |
|----------|---------|---------|---------|
| Source types | Secondary only | Mix primary + secondary | Primarily primary |
| Citation tracing | No | Key papers only | Systematic |
| Methodology critique | No | Awareness | Deep engagement |
| Competing frameworks | Mention | Compare | Evaluate with evidence |
| Original synthesis | Summarize | Explain relationships | Argue a position |
| Search documentation | Not required | Recommended | Required (inclusion/exclusion criteria) |
