# Plan 23-01: Fix Workflow Init Subcommand Names

## Goal

Update all 8 workflow files that call old init subcommand names to use the research-native names registered in `grd-tools.cjs`.

## Context

Phase 17 renamed init subcommands in `grd-tools.cjs` (e.g., `execute-phase` → `conduct-inquiry`, `plan-phase` → `plan-inquiry`) but the workflow `.md` files that call these subcommands were not updated. Every affected workflow hits `Error: Unknown init workflow` at runtime.

## Tasks

**Files to modify:**

| File | Line | Old Call | New Call |

| `grd/workflows/plan-inquiry.md` | 18 | `init plan-phase` | `init plan-inquiry` |

| `grd/workflows/verify-inquiry.md` | 30 | `init verify-work` | `init verify-inquiry` |

| `grd/workflows/complete-study.md` | 597 | `init execute-phase` | `init conduct-inquiry` |

**Action:** For each file, replace the old init subcommand name with the correct one. Only change the subcommand argument — do not modify paths or other parts of the line.
