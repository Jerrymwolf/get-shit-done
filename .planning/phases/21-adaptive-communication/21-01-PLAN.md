    XML mode (format='xml'):
    - stripTierContent('<tier-guided>hello</tier-guided>\n<tier-standard>world</tier-standard>\n<tier-expert>!</tier-expert>', 'guided', 'xml') keeps "hello", removes "world" and "!"
    - stripTierContent('<tier-guided>hello</tier-guided>\n<tier-standard>world</tier-standard>\n<tier-expert>!</tier-expert>', 'standard', 'xml') keeps "world", removes "hello" and "!"
    - stripTierContent('<tier-guided>hello</tier-guided>\n<tier-standard>world</tier-standard>\n<tier-expert>!</tier-expert>', 'expert', 'xml') keeps "!", removes "hello" and "world"
    - Unwrapped content preserved: 'before\n<tier-guided>G</tier-guided>\nafter' with tier='expert' yields 'before\nafter'
    - Multiline content inside blocks preserved: '<tier-guided>line1\nline2\nline3</tier-guided>' with tier='guided' yields 'line1\nline2\nline3'
    - Nested XML inside tier blocks: '<tier-guided>Use <src> tags for citations</tier-guided>' with tier='guided' yields 'Use <src> tags for citations'

    Comment mode (format='comment'):
    - stripTierContent('<!-- tier:guided -->\nhello\n<!-- /tier:guided -->\n<!-- tier:standard -->\nworld\n<!-- /tier:standard -->', 'guided', 'comment') keeps "hello", removes "world"
    - stripTierContent('## Heading\n<!-- tier:guided -->\nguidance\n<!-- /tier:guided -->', 'expert', 'comment') keeps "## Heading", removes guidance
    - Existing non-tier HTML comments preserved: '<!-- normal comment -->' stays for all tiers

    Validation:
    - stripTierContent(content, 'beginner', 'xml') throws Error with message containing "Invalid tier"
    - VALID_TIERS exports ['guided', 'standard', 'expert']

    Cleanup:
    - Three or more consecutive newlines reduced to exactly two
    - No orphaned tier tags in output for any tier (round-trip safety)
