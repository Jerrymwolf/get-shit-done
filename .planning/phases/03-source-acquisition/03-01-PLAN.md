Create grd/bin/lib/acquire.cjs:

**detectSourceType(url)** — Analyze URL to determine source type and preferred acquisition method:
- arxiv.org/pdf/* -> { type: 'pdf', ext: 'pdf', methods: ['wget', 'web_fetch'] }
- arxiv.org/html/* -> { type: 'md', ext: 'md', methods: ['firecrawl', 'web_fetch'] }
- github.com/*/issues/* -> { type: 'md', ext: 'md', methods: ['gh-cli', 'web_fetch'] }
- github.com/*/* (README) -> { type: 'md', ext: 'md', methods: ['firecrawl', 'web_fetch'] }
- *.pdf URL -> { type: 'pdf', ext: 'pdf', methods: ['wget', 'web_fetch'] }
- Default -> { type: 'md', ext: 'md', methods: ['firecrawl', 'web_fetch', 'wget'] }

**acquireSource(options)** — Acquire a source file with fallback chain:
```
options: {
  url: string,           // Source URL
  slug: string,          // Descriptive slug for filename
  sourcesDir: string,    // Directory to save file into
  toolRunner?: Function, // Injected for testing (default: real execFileSync)
}
```

Implementation:
1. Call detectSourceType(url) to get type info and method order
2. Generate filename using vault.cjs generateSourceFilename(slug, ext)
3. Try each method in order:
   - firecrawl: execFileSync('firecrawl', ['scrape', url, '--format', 'markdown'])
   - web_fetch: This is a Claude tool, so for the module just call toolRunner with 'web_fetch' — in real execution, the calling agent handles this
   - wget: execFileSync('wget', ['-q', '-O', outputPath, url])
   - curl: execFileSync('curl', ['-sL', '-o', outputPath, url])
   - gh-cli: execFileSync('gh', ['issue', 'view', issueNum, '--repo', repo, '--json', 'body'])
4. If method succeeds, save content to sourcesDir/filename, return success
5. If method fails, log attempt and try next
6. If all fail, return { status: 'unavailable', reason: aggregated error messages }

**updateSourceLog(sourceLogPath, entry)** — Append a row to SOURCE-LOG.md:
```
entry: { source, url, method, file, status, notes }
```
Read file, find table, append row after last `|` line, write back.

Use execFileSync (not exec) for all external tool calls — accept toolRunner injection for testing.
Import generateSourceFilename from vault.cjs.
