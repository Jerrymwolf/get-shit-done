Add to acquire.cjs:

**extractReferences(noteContent)** — Parse a note's References section and extract referenced filenames:
- Find ## References section using regex
- Extract backtick-wrapped filenames matching pattern: `name_YYYY-MM-DD.ext`
- Extract wikilink references: [[Note-Name]]
- Return { files: [...], notes: [...] }

**validateReferences(noteContent, sourcesDir, sourceLogPath)** — Check that every referenced file exists:
- Parse references using extractReferences()
- For each file reference, check if it exists in sourcesDir using fs.access()
- If missing, check SOURCE-LOG.md for documented unavailable status (not a real gap)
- Scan sourcesDir for orphan files (exist but not referenced, excluding SOURCE-LOG.md)
- Return { valid, missing, orphans, referenced, noteRefs }

Both functions use only Node built-ins (fs, path). No external dependencies.
Use execFileSync for any external tool calls (never exec with shell interpolation).
Export both functions from module.exports.
