# Phase 31 Plan 01: Reconceptualize Export and Presentation Commands

## Objective

Reconceptualize four output-oriented PM commands into research delivery tools: ship becomes export-research, pr-branch becomes export-clean, ui-phase becomes presentation-design, ui-review becomes output-review. Each command's workflow file and CLI route are updated end-to-end with research-native content.

## Context

- Phase 27 renamed command identifiers but left PM-oriented content
- Phase 28 reoriented agent prompts to acknowledge research outputs
- Phase 29 created research template variants
- This phase rewrites the workflow/command CONTENT to be research-native

## Tasks
